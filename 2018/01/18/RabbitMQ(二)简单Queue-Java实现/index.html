RabbitMQ(二)简单Queue-Java实现

发表于 2018-01-18 | | 阅读次数:

RabbitMQ 是实现AMQP（高级消息队列协议）的消息中间件的一种，用于在分布式系统中存储转发消息，在易用性、扩展性、高可用性等方面表现不俗。

RabbitMQ 有次特性，那么我们就简单的利用他的特性，实现简单的生产者-消费者解耦。

准备

环境

  1. CentOS7.2 64
  2. RabbitMQ 3.6.9
  3. JAVA8

概念

初学不用讲整个过程搞得太复杂，先参考官方提供的DEMO。

https://www.rabbitmq.com/getstarted.html

首先，我们需要理解以下三个概念：

  • Producer–生产者，发送方
  • Consumer–消费者，订阅方
  • Queue–消息队列
